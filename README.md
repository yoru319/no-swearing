# 脏话屏蔽器
可以用于聊天时屏蔽自己打出的脏话，让自己更加文明。也可以让别人装上，吵架时对方就像打在棉花上，什么都说不出来。

（原理是利用espanso，即类似按键精灵，优点是在微信qq，等任意平台都可以使用，且不会被风控。缺点就是开启后，只要使用键盘就会自动检测屏蔽脏话的拼音，可能会影响日常使用。）

第一步：下载espanso。  网址：https://espanso.org/install/，下载自己电脑适配的版本。

下载后，右下角任务栏有一个白色图标即是开启。默认的下载位置是C:\Users\用户名\AppData\Roaming

第二步：下载我上传的base.yml。

第三步：打开C:\Users\用户名\AppData\Roaming\espanso\match，里面有个base.yml文件。将其用刚才下载的文件替换后，点击左上角保存即可运行。

我设置了一些常见的脏话作为屏蔽词，当你输入屏蔽词后会自动在聊天框被替换成“我喜欢你”等。

如果想修改，可以打开base.yml，找到

  - trigger: "屏蔽词"
  - 
    replace: "输出词"
    
将双引号中内容自行删减或修改成自己想要的词或句子。
